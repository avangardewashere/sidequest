# Plans

- Product/UI planning documents, trackers, and implementation checklists.
- Keep planning artifacts here unless they are cycle-execution records.

## Cycle Planning Convention

- Strategic multi-cycle roadmap docs live in `documentation/plans/cycles/`.
- Phase execution docs live in `documentation/plans/cycle-<n>/` as a pair:
  - `phase-<n>-<m>-<slug>-plan.md`
  - `phase-<n>-<m>-<slug>-tracker.md`
- Reusable authoring templates live in `documentation/plans/templates/`.

## Execution Rule

- Run one phase at a time.
- Do not start the next phase until the current phase tracker exit criteria are fully checked.
